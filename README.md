# SEO by Susmita

Personal freelance SEO services website — built with plain HTML & CSS, hosted on GitHub Pages.

## 🚀 How to publish on GitHub Pages

1. **Create a new GitHub repository**
   - Go to [github.com](https://github.com) → click **New repository**
   - Name it `seo-by-susmita` (or anything you like)
   - Set it to **Public**
   - Click **Create repository**

2. **Upload the files**
   - Click **Add file → Upload files**
   - Drag and drop `index.html` and this `README.md`
   - Click **Commit changes**

3. **Enable GitHub Pages**
   - Go to your repo → **Settings** → **Pages** (left sidebar)
   - Under **Source**, select **Deploy from a branch**
   - Branch: `main` / folder: `/ (root)`
   - Click **Save**

4. **Your site is live!** 🎉
   - In a minute or two, visit: `https://YOUR-GITHUB-USERNAME.github.io/seo-by-susmita`
